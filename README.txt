## Installation

#### Prerequisites
- CMake >= v.3.10
- C++ Compiler >= C++17
- Boost C++ Library

Before compiling and running this project, Boost must be installed in the following folder: /third_party/boost.

#### Install
```bash
mkdir build
cd build
cmake -S .. -B .
make
```

## How to use

#### Run
```bash
./weisfeiler-leman
```
